Hai! Repo ini basically proyek sederhana tapi keren buat nganalisis ulasan pemain game Honkai: Star Rail di Google Play Store pake teknik NLP (Natural Language Processing). Gue fokus ke sentimen ulasan dari user Indonesia, mulai dari ngumpulin data sampe bikin model prediksi sentimen positif/negatif. Ide utamanya buat liat gimana respon komunitas terhadap update game, seperti rilis karakter baru atau event-event besar. Semuanya dibangun pake Jupyter Notebook, dan kode-nya gampang diikuti buat yang lagi belajar data science atau pengen eksperimen sendiri. Kalo lo fans Honkai atau suka main-main data, ini bisa jadi starting point bagus buat project serupa.

Apa yang Dibahas di File Kode? File utama "NLP_Sentiment_Analysis_Honkai_Star_Rail_Reviews.ipynb" ngebahas alur lengkap analisis sentimen, dari instal library sampe deployment model. Mulai dari scraping ulasan pake google-play-scraper dengan filter tanggal spesifik (misalnya November-Desember 2025), biar data-nya fresh dan relevan sama update game terkini.
Analisis yang Dilakukan: Gue analisis sentimen ulasan buat ngerti opini pemain soal game-nya, kayak apakah mereka suka cerita, gameplay, atau complain soal bug/gacha. Pake IndoBERT dari Hugging Face buat labelling otomatis (positif/negatif/netral), terus train model Naive Bayes pake TF-IDF buat prediksi. Ada juga visualisasi wordcloud buat liat kata-kata kunci yang sering muncul di ulasan positif vs negatif.
Teknik dan Tools yang Dipake: Preprocessing teks pake regex buat bersihin, stemming dengan Sastrawi biar handle bahasa Indonesia, plus scikit-learn buat machine learning dasar. Semuanya di-wrap pake tqdm biar prosesnya keliatan progress-nya, dan transformers buat model BERT yang lebih advanced.
Kenapa Ini Berguna? Hasilnya bisa kasih insight buat developer game atau komunitas, misalnya ngerti kenapa rating drop setelah update tertentu. Kode-nya modular, jadi lo bisa modifikasi buat game lain atau tambah fitur seperti analisis lebih dalam pake deep learning.
